# 🚀 Projeto Pipeline de Dados

Este projeto foi desenvolvido como trabalho final do curso de Pipeline de Dados da Alura, com o objetivo de aplicar na prática conceitos fundamentais de engenharia de dados, como ingestão, transformação e organização de dados.

## 📌 Objetivo

Construir um pipeline de dados eficiente que seja capaz de:

Realizar o tratamento e transformação dos dados
Organizar os dados de forma estruturada
Preparar os dados para análise ou uso futuro

## 🛠️ Tecnologias utilizadas
- Python
- Pandas
- Jupyter
- Git & GitHub

## ⚙️ Funcionalidades

✔️ Tratamento e limpeza de dados

✔️ Estruturação em DataFrames

✔️ Exportação/armazenamento dos dados

✔️ Organização modular do código

## 🔄 Pipeline de Dados

O fluxo do projeto segue as etapas clássicas de um pipeline ETL:

Extração (Extract)

- Coleta de dados JSON e CSV previamente baixados.

Transformação (Transform)

- Limpeza, tratamento e organização dos dados utilizando Python e Pandas.

Carga (Load)

- Armazenamento dos dados tratados para uso posterior.

## 📊 Resultados

O projeto gera dados tratados e organizados, prontos para análise ou integração com outras aplicações, demonstrando na prática a construção de um pipeline de dados eficiente.

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram reforçados conceitos como:

- Construção de pipelines de dados
- Processos de ETL
- Manipulação de dados com Pandas
- Boas práticas de organização de código
